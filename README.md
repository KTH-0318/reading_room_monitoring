# reading_room_monitoring

라즈베리파이 + YOLOv8n + 환경센서로 열람실 좌석의 실사용 여부와 온·습도·불쾌지수·미세먼지를
실시간 웹 대시보드로 제공한 IoT 모니터링 시스템.
**얼굴 대신 책상 위 물건을 탐지**해 프라이버시 부담을 줄였고, **실제 열람실 72석에 배포·시연**했다.

## 시스템 구성

```
라즈베리파이 (DHT11 온습도 · 미세먼지 센서 · 웹캠 3대)
      │  센서값 · 탐지 결과 전송
      ▼
   MySQL DB
      │
      ▼
 Python 대시보드 ──▶ 웹 배포 (관리자 / 이용자)
```

- 좌석 점유 판정: 책상 구역 분할 → 물건(노트북·책·펜 등) 5초 이상 감지 시 "사용 중" — 얼굴 인식 없이 점유만 추정
- 모델: 라즈베리파이 메모리 제약에 맞춘 최경량 **YOLOv8n**
